# Our Valentine's Memory Game ❤️

A personalized digital experience created for a very special person.

## Features

- **Memory Match:** A custom card game featuring our favorite photos.
- **Heart-Shaped Grid:** A unique layout designed for Valentine's Day.
- **Surprise Proposal:** A special interactive sequence once the game is complete.

## How it was built

This project is built using **Next.js** and **Tailwind CSS**. I customized the logic to handle my own photo collection and optimized the experience for mobile and desktop.

## Running Locally

1. Clone the repo
2. Install dependencies: `npm install`
3. Start the server: `npm run dev`
